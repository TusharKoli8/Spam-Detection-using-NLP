***NLP-Based SMS Spam Detection***


A simple NLP project to classify SMS messages as Spam or Ham using text preprocessing and TF-IDF.

-- Features
Text cleaning & preprocessing
TF-IDF feature extraction
Spam keyword analysis
Rule based prediction system
Streamlit UI for live predictions

-- Run the Project

**python -m src.main**


streamlit run app.py(optioal)

 Tech Stack

Python, NLP, Scikit-learn, TF-IDF, Streamlit

-- Conclusion

An end to end spam detection system demonstrating core NLP concepts with a simple interactive interface.
